    - Title pattern as editable Input (onBlur saves via useUpdateTemplate)
    - Content type badge (text/carousel)

    - "Delete" button with confirm dialog (window.confirm)

  - After generation: show success message with "Switch to Month view" link

**2. Create src/components/pipeline/AnalyticsStrip.tsx:**

Compact analytics strip rendered below the page header, above the view content.

- Props: `{ items: ContentReview[], currentMonth: Date }`
- Computes from items filtered to currentMonth:
  - total: items with scheduled_date in current month
  - posted: items with status='posted'
  - pending: items with status='pending'
  - approved: items with status='approved'
  - draft: items with status='draft'
- Renders as a single-row flex strip with bg-[hsl(var(--bg-elevated))] and rounded-[var(--radius-md)] padding:
  - "This month:" label in muted text
  - Stat pills: "{total} planned" | "{posted} posted" | "{pending} pending" | "{draft} draft" -- each as a compact badge-like span
  - Cadence indicator per D-26: compute posts per week in current month. Target is 6/week.
    - Green (on track): >= 5 posts/week average
    - Amber (falling behind): 3-4 posts/week
    - Red (significant gaps): < 3 posts/week
    - Display as a small colored dot + "Cadence: on track" / "falling behind" / "gaps detected"
  - Gap alert per D-27: scan scheduled dates for 2+ consecutive days with no content. If found: show amber warning icon + "Gap: {date range} has no content"
- Strip is collapsible: small chevron toggle. Default expanded. Uses Framer Motion for collapse animation (height: 0 -> auto).
