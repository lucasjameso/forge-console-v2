Card migration pattern (from RESEARCH.md):
- Find: `className="rounded-lg border bg-card p-6 shadow-card"` (or similar with className containing rounded-lg border bg-card)
- Replace with: `<Card className="p-6">` (simple card) or `<Card className="p-6 transition-shadow hover:shadow-card-hover cursor-pointer">` (interactive card)
- Import: `import { Card } from '@/components/ui/card'`
- IMPORTANT: Do NOT use CardContent/CardHeader for simple cards. Put padding directly on Card.
- Interactive cards (ones that navigate or have onClick): add `hover:shadow-card-hover` (per D-07)
- Static display cards (stat cards, system health): NO hover shadow

Button migration pattern (from RESEARCH.md):
- Coral primary: `<button className="inline-flex items-center gap-1.5 rounded-md bg-coral px-4 py-2 text-sm font-medium text-white...">` --> `<Button>`
- Ghost/outline: `<button className="inline-flex items-center gap-1.5 rounded-md border border-input bg-transparent px-4 py-2...">` --> `<Button variant="outline">`
- Destructive: style with status-error bg --> `<Button variant="destructive">`
- Import: `import { Button } from '@/components/ui/button'`
- If button has an icon + text, the Button component already has gap-2 in its base class
- Pass `size="sm"` for smaller buttons if the original used smaller padding

Typography class mapping (per D-12):
| Inline fontSize | Maps to class | Tailwind fallback |
|-----------------|---------------|-------------------|
| 36px | text-stat | - |
| 28px | text-page-title | - |
| 20px | (one-off) | text-xl |
| 18px | text-section-header | - |
| 15px | text-card-title | - |
| 14px | text-body | - |
| 13px | text-body-sm | - |
| 12px | text-caption | - |
| 11px | text-overline OR text-[11px] | text-[11px] for non-uppercase |

Spacing rules (per D-15):
- Section gaps (between major sections on a page): 32-40px --> use `space-y-8` (32px) or `space-y-10` (40px) or `gap-8`/`gap-10`
- Card gaps (between cards in a grid or list): 20-24px --> use `gap-5` (20px) or `gap-6` (24px)

Row padding rules (per D-08, D-09):
- Compact rows (activity log, system health, kanban items): `py-2 px-3` with `gap-1.5` (6px)
- Normal rows (brain dump tasks, notes, settings items): `py-3 px-4` with `gap-2.5` (10px)
